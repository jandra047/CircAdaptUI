Welcome to CircAdaptUI documentation
====================================

CircAdaptUI is a graphical user interface (GUI) for the 
`CircAdapt <www.circadapt.org>`_ framework. It uses the CircAdapt framework
to simulate cardiovascular hemodynamics and provides immediate visualization 
of the results, enabling users to interact with and analyze simulations in 
real-time.

########
Key features
########

#. Real-time simulation 

   * Incremental, step-by-step simulations.
#. Real-time visualization

   * Instant display of simulation results.
#. Real-time parameter adjustment

   * Interactive parameter modification with immediate effects.
